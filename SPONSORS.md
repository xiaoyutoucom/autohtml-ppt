### 作者如何开通 GitHub Sponsors（一次性）

1. 浏览器登录 GitHub 账号 **xiaoyutoucom**  
2. 打开 https://github.com/sponsors/xiaoyutoucom （或 Settings → **Sponsors**）  
3. 按向导申请 **GitHub Sponsors**（需绑定收款方式，如 Stripe；国内账号以页面提示为准）  
4. 审核通过后，上面的 Sponsors 链接即可收款  
5. （推荐）仓库 About 勾选「Sponsorships」，README 已链到本页  

> 我无法替你点开通；必须用你的 GitHub 账号在网页完成。开通后一般不用再改仓库链接。
