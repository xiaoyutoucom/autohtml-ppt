Anthropic · Long-running Harness

  • 痛点：Agent 自评「Done」，UI/API 却一塌糊涂
  • 做法：拆任务的 Initializer + 逐步实现的 Coding Agent
  • 再加 Generator–Evaluator：别让「写代码的人」给自己打分
  • 启示：长任务可靠性来自角色分工与验收环，不是更长 prompt

Ralph Pattern · 自主过夜

  • 核心：PRD 条目化 + 循环调用 Claude Code / Amp 等直到清零
  • Harness 提供：目标清单、失败重试、产物落盘、停止条件
  • 价值：演示「无人值守」不是玄学，是可复制的运行时模式
  • 边界：仍需验收与护栏，否则会「假完成」
