# Contributing

## Before you start

1. Read [`docs/skills/tech-training-deck/SKILL.md`](docs/skills/tech-training-deck/SKILL.md).  
2. Edit **`docs/deck/`**, then run `python tools/build_deck.py`.  
3. Do not commit secrets, customer data, or unlicensed media.

## Pull requests

1. Sign the [CLA](./CLA.md) in the PR description.  
2. Keep changes focused (one concern per PR).  
3. Rebuild if you touch deck source; smoke-test `docs/harness_training.html`.  
4. Prefer English commit messages; Chinese OK for content/docs.
